# TimeFlow - Tableau de Bord de Gestion du Temps

Application React Native/Expo avec un design style tableau de bord de voiture de sport.

## Fonctionnalités

- Deux jauges circulaires (demi-cercles) pour les périodes 
- Affichage du temps restant en temps réel (Secandes)
- Design Dark Mode avec couleurs Cyan (matin) et Orange (après-midi)
- Animations fluides avec l'API Animated de React Native

## Installation

1. **Installer les dépendances :**
   ```bash
   npm install
   ```
   ou
   ```bash
   yarn install
   ```

2. **Lancer l'application :**
   ```bash
   npm start
   ```
   ou
   ```bash
   yarn start
   ```

3. **Choisir la plateforme :**
   - Appuyez sur `a` pour Android
   - Appuyez sur `i` pour iOS
   - Appuyez sur `w` pour Web
   - Scannez le QR code avec l'app Expo Go sur votre téléphone
